# プライバシーポリシー

最終更新日: 2026年4月10日

## はじめに

「おサボり番人」（以下「本アプリ」）は、Kakeru Nakamura（以下「開発者」）が提供する集中支援アプリです。本プライバシーポリシーは、本アプリにおける個人情報の取扱いについて説明します。

## 収集する情報

本アプリは、**個人情報を一切収集しません。**

本アプリが取り扱うデータは以下の通りですが、すべてお使いのデバイス内にのみ保存され、外部に送信されることはありません。

| データ | 用途 | 保存場所 |
|--------|------|---------|
| やることタイトル | 集中セッションの表示 | デバイス内（App Groups） |
| 集中時間の設定 | タイマー管理 | デバイス内（App Groups） |
| 寄り道回数 | セッション結果の表示 | デバイス内（App Groups） |
| 制限対象アプリの選択情報 | アプリの制限管理 | デバイス内（App Groups） |

## データの送信・共有

- 本アプリは、外部サーバーへのデータ送信を**一切行いません。**
- 第三者へのデータ提供・共有は**一切行いません。**
- 広告配信サービス・アクセス解析ツールは**使用していません。**

## Screen Time API の使用について

本アプリは、Apple の Screen Time API（Family Controls / ManagedSettings / DeviceActivity）を使用しています。これらの API は、ユーザー自身が設定した集中セッション中に、選択したアプリの使用を一時的に制限する目的でのみ使用されます。

- ユーザーの操作により認可を取得します（ペアレンタルコントロールではありません）
- アプリの使用履歴や統計情報を記録・送信することはありません
- 制限はセッション終了時に自動的に解除されます

## お子様のプライバシー

本アプリは、13歳未満のお子様を対象としたアプリではありません。13歳未満のお子様から意図的に個人情報を収集することはありません。

## データの削除

本アプリのデータはすべてデバイス内に保存されています。アプリを削除することで、関連するすべてのデータが削除されます。

## プライバシーポリシーの変更

本プライバシーポリシーは、必要に応じて更新されることがあります。変更があった場合は、本ページにて通知します。
